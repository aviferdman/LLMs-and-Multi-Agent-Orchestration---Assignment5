# Run Experiment 1 Command

Executes the "Lost in the Middle" experiment.

## What It Does

1. Checks if experiment1.py is implemented
2. Runs experiment with all position variations (start, middle, end)
3. Collects data for statistical analysis
4. Saves results to `results/raw/experiment1/`
5. Logs progress to `agents_log.txt`

## Usage

Say: **"Run Experiment 1"** or **"Execute Lost in the Middle experiment"**

## Prerequisites

- Infrastructure set up (check agents_log.txt)
- Experiment 1 implemented
- Ollama running

## Expected Outcome

- Raw data collected in results/raw/experiment1/
- Progress logged
- Ready for statistical analysis
